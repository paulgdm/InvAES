# AES

Decryption with AES 128bits in C

## Compilation

In a shell bash :

```bash
make
```

## Usage

```bash
sh decrypt.sh file1.txt file2.txt
```

with :
* file1 : ciphertext
* file2 : cleartext

## Exemple

```bash
sh decrypt.sh ciphertext.txt cleartext.txt
```
